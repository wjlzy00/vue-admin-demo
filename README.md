# Vue3 + Antd 电商后台 Demo

> 一个面向初学者的最小可运行电商后台骨架，持续迭代 8 大模块。

## 预览
- 在线 Demo: https://vue-admin-demo.vercel.app  

## 技术栈
- Vue 3 + Vite
- Ant Design Vue 4.x
- Vue Router 4
- Pinia (状态管理)
- Axios (请求)
- ECharts (图表)
